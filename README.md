# 화물차 사고 데이터 분석 대시보드

이 프로젝트는 화물차 사고 데이터를 분석하고 시각화하는 대시보드입니다.

## 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 대시보드 실행:
```bash
python dashboard.py
```

3. 웹 브라우저에서 `http://localhost:8050` 접속

## 주요 기능

- 데이터 처리 파이프라인 시각화
- 다양한 데이터셋 선택 (화물차 사고, 차종별 교통사고, 사망사고 및 휴게소)
- 시간별/지역별/사고 유형별 분석
- 커스터마이즈 가능한 시각화 옵션

## 데이터 처리 파이프라인

1. 원본 데이터 로드
2. 데이터 전처리
3. 분석 수행
4. 시각화 생성

## 시각화 옵션

- 트렌드 라인
- 평균값 표시
- 비교 분석
